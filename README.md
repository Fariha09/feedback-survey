# Feedback Survey Application

## Overview
A responsive web-based survey form built using JavaScript, HTML5, and CSS3. This project demonstrates client-side validation, structured form handling, and user interaction logic.

Live Demo: https://fariha09.github.io/feedback-survey/

## Features
- User input fields with validation
- Real-time form validation feedback
- Conditional rendering based on user responses
- Responsive layout across devices

## Technologies Used
- JavaScript (ES6+)
- HTML5
- CSS3
- GitHub Pages (Deployment)

## Technical Highlights
- Implemented client-side validation logic
- Managed form state and input handling
- Structured conditional rendering for dynamic user experience
- Ensured cross-browser compatibility

## What I Learned
- Building accessible and user-friendly form interfaces
- Handling structured user input and validation logic
- Improving UX through responsive design principles
